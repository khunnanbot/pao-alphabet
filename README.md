## Pa'O Alphabet Interactive Poster

An interactive educational poster for learning the Pa'O alphabet (ပအိုဝ်ႏ ဗီး) with 33 characters and numbers (0-9).

### Features

- **Interactive Learning**: Click on any letter or number to hear pronunciation
- **Responsive Design**: Works on desktop, tablet, and mobile devices
- **Beautiful UI**: Color-coded alphabet cards with gradient backgrounds
- **Download Option**: Save the poster as an image

### Characters Included

- **Alphabet**: 33 Pa'O letters (ပအိုဝ်ႏ ဗီး ၃၃-ဖြုံႏ)
- **Numbers**: 0-9 (ပအိုဝ်ႏ ဂဏန်ꩻ)

### How to Use

1. Visit the website
2. Click on any alphabet character or number
3. Listen to the pronunciation
4. Click "Save as Image" to download the poster

### Technical Details

- Built with HTML5 and CSS3
- No external dependencies required
- Audio files can be added to the `/audio/` directory
- Fully responsive grid layout
